## About SysCord
This bot acts as an async communication bridge between Discord and your Nintendo Switch using sys-botbase.

### Features:
    - Remote handling and debugging
    - Screenshot uploading directly to PC

### Requirements:
    1. Python 3.8
    2. CFW Nintendo Switch
    3. sys-botbase 2.0+
